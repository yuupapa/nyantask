# 猫画像 生成ガイド

## 概要
にゃんタスクで使用する猫イラスト100枚を AI 画像生成で作成するためのガイド。
画像は `public/cats/cat-001.png` 〜 `cat-100.png` に配置する。

## 画像仕様
- **サイズ**: 512×512px（アプリ内で縮小表示）
- **形式**: PNG（透過背景）
- **スタイル**: ゆるい・かわいい・デフォルメ（ねこあつめ風）
- **ポーズ**: 正面やや斜めの座りポーズ（統一感を出すため）
- **背景**: 透明（transparent）

## 共通プロンプト（ベース）

```
A cute, simple, chibi-style cartoon cat sitting in a relaxed pose, 
front-facing, round body, small legs, big head proportions.
Neko Atsume / kawaii illustration style.
Flat colors, clean lines, soft pastel shading.
Transparent background, PNG.
512x512 pixels.
```

## 100匹のバリエーション

以下の属性を組み合わせて100匹の個性を出す。

### 毛色・柄（20種）
1. 三毛（白+オレンジ+黒パッチ）
2. 黒猫（全身黒、黄色い目）
3. 白猫（全身白、ピンクの鼻）
4. キジトラ（茶色+黒の縞模様）
5. サバトラ（グレー+黒の縞模様）
6. 茶トラ（オレンジの縞模様）
7. ハチワレ（白+黒、顔が左右で色分け）
8. ぶち（白地に黒い斑点）
9. シャム（クリーム地+こげ茶の顔・耳・足）
10. ロシアンブルー（ブルーグレー）
11. スコティッシュ（折れ耳、ベージュ）
12. マンチカン（短足、クリーム色）
13. ペルシャ（長毛、白）
14. アメリカンショートヘア（シルバータビー）
15. ベンガル（ヒョウ柄）
16. ラグドール（白+グレーポイント）
17. メインクーン（大きめ、茶色長毛）
18. ブリティッシュショートヘア（灰色、丸顔）
19. ソマリ（赤茶、ふさふさ尻尾）
20. トンキニーズ（ミンク色）

### 目の表情（5パターン）
- まんまる目（デフォルト、かわいい）
- つり目（キリッと）
- たれ目（おっとり）
- 笑い目（にっこり閉じ目）
- キラキラ目（ハイライト多め）

### アクセサリー（ランダムに付与、なしも可）
- リボン（赤、青、ピンク、黄色）
- 首輪（ベル付き）
- 花冠
- 蝶ネクタイ
- バンダナ
- メガネ
- 帽子（ベレー帽、麦わら帽子）
- マフラー
- なし（プレーン）

## 生成の進め方

### ChatGPT (DALL-E) で生成する場合

1セッションで5〜10枚ずつ生成するのが効率的。

**プロンプト例（1匹ずつ）:**
```
Generate a cute chibi cartoon cat illustration.
- Cat type: 三毛 (calico) with white, orange, and black patches
- Expression: big round eyes, happy smile
- Accessory: small red ribbon on the ear
- Style: Neko Atsume / kawaii, flat colors, clean outlines
- Pose: sitting, facing slightly left, relaxed
- Background: transparent (PNG)
- Size: 512x512
```

**バッチ指示（ChatGPTに）:**
```
これから猫のイラストを100枚生成します。
以下のスタイルで統一してください：
- ねこあつめ風のゆるくてかわいいデフォルメ猫
- 座りポーズ、正面〜やや斜め
- 透過背景 PNG、512×512
- フラットな色使い、クリーンな線

1匹ずつ毛色・柄・表情・アクセサリーを変えて個性を出します。
まず最初の10匹をお願いします：

1. 三毛猫、まんまる目、赤リボン
2. 黒猫、つり目、ベル付き首輪
3. 白猫、たれ目、花冠
4. キジトラ、笑い目、アクセなし
5. サバトラ、まんまる目、バンダナ
6. 茶トラ、キラキラ目、蝶ネクタイ
7. ハチワレ、まんまる目、アクセなし
8. シャム、つり目、青リボン
9. ロシアンブルー、たれ目、メガネ
10. スコティッシュ（折れ耳）、笑い目、ベレー帽
```

### 生成後の作業

1. 画像をダウンロード
2. `cat-001.png` 〜 `cat-100.png` にリネーム
3. `public/cats/` に配置
4. `npm run dev` で確認

## ファイル配置

```
public/
  cats/
    cat-001.png   ← 三毛猫、まんまる目、赤リボン
    cat-002.png   ← 黒猫、つり目、ベル付き首輪
    cat-003.png   ← 白猫、たれ目、花冠
    ...
    cat-100.png
```

## 注意事項
- 透過背景が重要（アプリ内で背景色が変わるため）
- サイズは統一（512×512）
- あまりリアルすぎない、ゆるいタッチで
- 画像が揃うまではSVGフォールバックが表示される
